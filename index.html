This tool provides a way to determine what blueprints you can build using the provided assets.

It shows both direct and derived matches. Derived matches are those blueprints which require items produced by other blueprints.

Please, note that this tools uses only material types. It doesn't take into account the amount of materials and skills required to use a blueprint.
